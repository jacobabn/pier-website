FAQ

Questions, answered

Which AI models can I use?

Three options, switchable from Settings or per-connection: Anthropic (Claude Opus 4.5, Sonnet 4.5, Haiku 4.5 — pay-per-token), OpenAI (GPT-5, GPT-4o, o3, o4-mini — pay-per-token), or Claude Code (use your existing €200 Max subscription at €0 per chat turn). Bring your own key, Pier never sees it leave your Mac. Need a different provider? The architecture is pluggable — Gemini, DeepSeek, and Mistral support is on the roadmap.

Do I need an API key to start?

Only if you want to use Anthropic or OpenAI. If you have Claude Code installed (free with any Anthropic Max subscription), Pier routes through that at zero per-chat cost. No API key required. For everyone else: grab a key at console.anthropic.com or platform.openai.com, paste in Settings, done.

Will it actually edit my live website?

Yes — that's the whole point. Pier connects via FTP/SFTP to any host (DirectAdmin, cPanel, Plesk, custom). Every edit is proposed as a diff first — you click Accept, Pier uploads, and the file's previous state is saved as a version you can restore at any time.

Is my FTP password safe?

Yes. Credentials are encrypted (AES-256-GCM) on disk with HMAC integrity checks. Nothing leaves your Mac except the FTP traffic itself. We never see your passwords. The encryption key sits in your local .env with owner-only file permissions and benefits from FileVault disk encryption.

What if the AI breaks my site?

Two safety nets: (1) every edit is a diff you approve before applying, (2) every accepted edit creates a restore point. For high-risk files (.htaccess, wp-config.php, composer.json) Pier auto-creates a folder snapshot before applying. Worst case: one click and you're back to the previous state.

Does it work with WordPress?

Excellent with WordPress. Pier detects WP automatically, reads wp-config.php safely (secrets auto-redacted in chat), then offers to install a self-destructing PHP bridge so the AI can also browse and edit the database — wp_posts, wp_options, wp_users, custom plugin tables. You get phpMyAdmin-style cell editing plus AI-driven UPDATE/INSERT/DELETE proposals. Every change is reversible.

How does the database access actually work?

Pier installs a tiny PHP file (under a 16-character random folder name like .pier-a1b2c3d4/) inside your own webroot, behind dual-token auth (256-bit secret in both an HTTP header AND request body). It only accepts SELECT / SHOW / DESCRIBE — plus single-row UPDATE / INSERT / DELETE if you turn on "Enable edits" in the sidebar. The bridge auto-destructs after 10 minutes idle, or after 10 failed auth attempts. Your database password is sent over HTTPS per request, never stored on the bridge.

Will the AI mess with my live database without warning?

No. Mutations are always proposals. The AI can SELECT freely (read-only), but UPDATE / INSERT / DELETE always render as a card in chat showing before / after / SQL preview — you click Accept to actually run it. Bulk changes are deliberately not supported through the AI: one row at a time, by primary key, with prepared statements. If you want bulk SQL, use the Files tab's SQL console (still SELECT-only by default).

What about agencies and teams?

Each Pier seat handles unlimited connections. Add a license per developer; everyone gets their own chat history per client site. The €199 license covers 2 Macs (laptop + desktop). For agencies of 3-5+ people, just buy one license per seat. Team-shared knowledge base + shared connection pools are on the roadmap for late 2026.

Does it work offline?

The file editor works offline for locally cached content. The AI requires internet (to reach Anthropic or your local Claude). Version history and backups are stored on disk and work without a connection. License validation runs once on activation, then caches for 30 days — your laptop on a plane keeps working.

How does the licensing work? Can I move it to a new Mac?

Your license covers 2 Macs. Activate from each one's Settings → License. To move a license between machines, click "Deactivate this device" on the old Mac (frees the seat), then activate on the new one. Lost a Mac? Email support@abn.company and we'll free the seat from our side.

Because we hate subscriptions for tools we use occasionally. Pier should feel like Transmit, not like Adobe. You pay once, you own this version forever, you get bug fixes for 12 months. When v2 lands (~2027) with major new features, you can upgrade for ~€39 — only if you want to. The AI inference is BYO key, so we have no per-message cost to recover via subscription.

Why is it called Pier?

A pier is where boats dock to load and unload cargo. Your AI assistant docks with the FTP server to deliver and retrieve files. Short, memorable, and the metaphor actually works.
