05 —AI-driven database editor

Edit MySQL by chat. Every UPDATE is a proposal.

Pier auto-discovers your credentials from wp-config.php, .env, configuration.php — then installs a tiny self-destructing PHP bridge inside your own webroot.

Browse 67 tables. Edit cells like phpMyAdmin. Run SELECTs from a SQL console. Or just tell the AI: "set the site title to Northstar Bakery" — and approve the diff.

  • One-cell editing · primary-key bound · prepared statements

  • AI proposes mutations — never auto-fires UPDATE / INSERT / DELETE

  • Bridge auto-destructs after 10 min idle · IP-pinned · 256-bit token

  • HTTPS-only · dual-token auth · bulk mutations deliberately not supported
